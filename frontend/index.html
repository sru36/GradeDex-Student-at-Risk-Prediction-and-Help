About GradeDex

Academic project — ML-based Student at Risk Prediction & Help

🎓

Project Overview

GradeDex is an end-to-end machine learning web application for Student at Risk Prediction & Help. It turns structured student data into early risk signals, grade forecasts, and support-ready outputs that help educators act sooner.

🤖

Prediction Engine

The model pipeline learns ordered score boundaries and converts them into the fixed grade bands F, D, C, B, B+, A, AA, AAA. This keeps the prediction flow aligned with the grading rubric while also helping identify students drifting toward academic risk.

🩺

Missing Value Handling

Four columns have confirmed missing values: Teacher_Quality, Distance_from_Home, Parental_Education_Level, and Family_Income. All are imputed using mode (most frequent) strategy. Numeric features use median imputation for robustness against outliers.

⚖️

Why This Helps

GradeDex is built for action, not just reporting. It supports single-student prediction, bulk analysis, and intervention-oriented workflows so teachers can focus attention where support is most needed.

📊

Dataset

6,608 student records with 19 features covering academic habits, socioeconomic factors, and learning environment. Target variable: Exam_Score bucketed into F (<60), D (60-64), C (65-69), B (70-74), B+ (75-79), A (80-84), AA (85-89), and AAA (≥90).

🛠️

Tech Stack

Backend: Python, FastAPI, scikit-learn, pandas, numpy, joblib.
Frontend: Vanilla HTML/CSS/JavaScript, Chart.js.
Architecture: REST API + single-page application focused on prediction, risk surfacing, and student help flows.
